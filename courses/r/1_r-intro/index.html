R: Introductory course

Course contents

  • Day 1 (Introduction to data-analysis & R)

    In today’s world data analysis and visualization plays important roles in business analysis, product designs, defining strategies or policies etc. In every field, now data and its analysis help most importance. This lecture, will introduce you to data analysis, visualization and available tools and programming languages. R is simplest and widely used language in the field of data analysis and visualization. In this lecture we will also introduce you to R and its basic concepts. Why to do data analysis & visualization?

    • What are different tools available?
    • What is programming and basic concepts?
    • What & why R?
    • Installing R
    • Environment variables (a little talk on OS architecture)
    • Understanding R and R studio
    • R packages and sources
    • Basic syntax

  • Day 2 (Data-types and data-structures)

    In every language, data type and data structures are the basic way to store data in different objects. While learning programming languages, it is very important to understand how our language stores, different types of data. In this lecture, we will see basic data types and complex data structures in R. We will also see, the variables and different assignment operators to save data in named variable or object.

    • Data types in R
      • Logical
      • Numeric
      • Integer
      • Complex
      • Character
      • Raw
      • Strings
    • Data structures in R
      • Vectors
      • Lists
      • Matrices
      • Array
      • Factors
      • Data-frames
      • Variables

  • Day 3 (More on Data-types and data-structures)

    In the last lecture we have seen different data-types and data-structures used in R. In this section we will deal with different functions used on different data-structures in R and converting data-types from one to another.

    • Conversions
      • Character to numeric (vice-versa)
      • Numeric to boolean (vice-versa)
      • Boolean to character (vice-versa)
    • Functions on vectors
    • Functions on lists
    • Functions on matrix and data-frames

  • Day 4 (Operators)

    In this lecture, we will dive into different types of operators which are used to do mathematical operations, comparisons or filtering data.

    • Assignment operator
    • Arithmetic operators
    • Relational operators
    • Logical operators

  • Day 5 (Index)

    Index is numerical representation of position of elements in data-structures. In this lecture, we will see the usage of index to subset/retrieve/replace/delete data.

  • Day 6 (Loops)

    Loops are used to perform/execute a code several times or until a particular condition is attained.

    • for loop
    • while loop

  • Day 7 (Decision making (if…else))

    In every programming language a user may want to take decisions depending on the conditions. If…else can be used to make such decisions. In this lecture we will dive into conditions (if…else) loop used in R to make decisions or interrupt loops.

    • if…else
    • which function

  • Day 8 (Conditional filtering of data)

    A user may want to filter the data, using some controlled conditions/values. In this lecture, we will see how we can filter data in R in different data-structures using logical statements or search for specific pattern in case of strings.

    • Vectors
    • Matrix/data-frames
    • grep

  • Day 9 (Data transformation and reshaping)

    Matrices and Data frames are two important data structures in R which are routinely used to store data. In this lecture, we will see more operations which can be used to change or manipulate (in mathematical terms, transformation/normalization) data present in matrices or data frames.

    • Dive into Matrices
    • Dive into Data-frames
    • Data reshaping
      • Joining columns and rows
      • Merging two data-frames

  • Day 10 (Functions)

    Sometimes, a user may want to execute a chunk of code several times on different data-sets and want to reuse the code. In such cases, we can save our code chunk in form of customized function and reuse these functions by sourcing the script.

    • arguments
    • functions
    • return value

  • Day 11 (Reading/writing data)

    Excel, csv or zip files are common ways to store data. In this lecture, we will see multiple ways to read/write data in files.

  • Day 12 (Plotting data (ggplot grammar))

    ggplot2 is most popular package in R to plot data in different ways. Here, in this section, we will see basic ggplot grammar and how to plot our data.

    • GGplot grammar
      • Bar chart
      • Pie chart
      • Box plot
      • Histogram
      • Scatter plot
      • Line graphs

  • Day 13 (Real life examples: ANOVA, t-test, PcoA)

    R is most commonly used for statistical analysis of data. In this lecture, we will see how to apply statistical test like ANOVA, t-test etc on our data. Sometime, multi-dimensional data like gene-expression data has to be plotted in 2 dimensions (PCoA) we will also see some ordination techniques used in R.

  • Day 14 (Revision)

    This day is reserved for revision of concepts, functions and techniques we learned so far.

  • Day 15 (Assignment)

    Finally, we will see some real life examples for analysis in R. I will also share some assignments to practice. - Someday (Assignment discussion) This day is reserved for discussion of problems you may face while solving assignments.
